API Overview

This demonstration showcases the Experimentation Platform API, which allows developers to implement A/B tests, multivariate tests, and feature flags in their applications.

Core Concepts

Concept Description API Endpoints
Experiments Tests that compare different variants to measure their performance against metrics GET /api/v1/experiments
POST /api/v1/experiments
Assignments Assignments of variants to users POST /api/v1/experiments/{id}/assign
Exposures Records when a user views or interacts with an experimental variant POST /api/v1/experiments/{id}/exposure
Metrics Measurable values that determine experiment success POST /api/v1/experiments/{id}/metric
GET /api/v1/metrics
Results Statistical analysis of how variants perform GET /api/v1/experiments/{id}/results

Check out the included demo applications to see these concepts in action. The demos use the actual API endpoints and work with the experiments loaded in the database from the seed script.
